## Key facts for PE correlation
- The directory contains **NO raw PE codes**. It contains the acquisition org tree + named programs.
- Join path: `programs_mentioned` / CPE name (here) ⟷ program name / CPE (J-books) → PE code.
- For PE codes + dollars + requester: use J-books (R/P-docs), SAR/MAR, USAspending/FPDS, and NDAA/Approps markup tables. See plan §6.

## Caveats
- Title strings occasionally truncate where the source line wrapped mid-title. Names are clean (98.8% have first+last + a real title).
- ~6 narrative edge-case "people" (e.g. foreign liaison officers, a press mention) are genuinely in the source.
- Photos exist only inside the 86 org charts, not as one headshot per person. Per-person cropping is a deferred enhancement.
